# 使用方法

このページでは、Chrome拡張機能「Slack メッセージ非表示」の使用方法について説明します。

## 基本的な使い方

1. Chromeブラウザで Slack のウェブアプリ（`https://app.slack.com/client/*`）を開きます

2. 拡張機能がインストールされていると、`:done:` などのリアクションがついたメッセージは自動的に半透明になります

3. 拡張機能のアイコンをクリックすると、表示モードが切り替わります：
   - **通常モード**（デフォルト）: `:done:` リアクションがついたメッセージは半透明（30%の不透明度）で表示
   - **集中モード**: `:done:` リアクションがついたメッセージは完全に非表示

## 表示モードの切り替え

拡張機能のアイコンをクリックするたびに、表示モードが切り替わります：

1. **通常モード** → **集中モード**
2. **集中モード** → **通常モード**

## 対応する絵文字リアクション

この拡張機能は、以下の絵文字リアクションが付いたメッセージを処理します：

- `:done:`
- `:done_1:`
- `:done-1:`
- `:done_ja:`
- `:done_en:`

これらのいずれかの絵文字リアクションがメッセージに付いていると、そのメッセージは現在の表示モードに応じて半透明または非表示になります。

## 動作の仕組み

1. Slackページを開くと、拡張機能は自動的に有効になります
2. チャンネルを切り替えたり、スクロールしたりしても、新しく表示されるメッセージに対して自動的に処理が適用されます
3. MutationObserverを使用してDOMの変更を監視しているため、リアルタイムでメッセージの表示状態が更新されます

## ヒントとコツ

- タスク管理にSlackを使用している場合、完了したタスクに `:done:` リアクションを付けることで、未完了のタスクに集中できます
- チーム内で情報共有する際に、既読のメッセージに `:done:` リアクションを付けることで、未読のメッセージを視覚的に区別できます
- 重要なメッセージには `:done:` リアクションを付けないようにすることで、常に表示されるようにできます
